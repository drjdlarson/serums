add_components(means, scalings, dof_lst, weights)[source]

Add Student’s t-distributions to the mixture.

Parameters:

  • means (list) – Each is a N x 1 numpy array of the mean of the distributions to add.

  • scalings (list) – Each is a N x N numpy array of the scale of the distributions to add.

  • dof_lst (list) – Each is a float representing the degrees of freedom of the distribution to add.

  • weights (list) – Each is a float for the weight of the distributions to add. No normalization is done.

Return type:

None.
